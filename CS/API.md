# API?

### Application Programming Interface

**응용 프로그램에서 소통을 하기 위한 접점**
**애플리케이션에서 데이터를 읽거나 쓰기 위해 사용하는 인터페이스**

### UI(User Interface)

**사용자가 소통을 하기 위한 접점**

즉, 무언가가 소통을 하기 위함이냐에 따라 UI, API로 나뉠 수 있음

### API 종류

- HTTP API(흔히 우리가 말하는 API로 인터넷 웹상에서 사용되는 API)
  - Private API
  - Public API(Open API)
- Java API
- Android API
