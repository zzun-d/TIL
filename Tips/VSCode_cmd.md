### Visual Studio Code

- 단축키
  - 폴더 닫기 : **ctrl + k,  f**
  - 설정 보기 : **ctrl + ,(콤마)**
  - 터미널 열기 : **ctrl + `(백틱)**
  - 새 파일 열기 : **ctrl + n**
  - 코드 복사 붙여넣기 : **alt + shift + up or down**
  - 여러 줄 한번에 작성 : **alt + ctrl + up or down**
  - 코드 순서 바꾸기 : **alt + up or down**
  - 같은 단어 찾기 : **ctrl + d**
  - Markdown preview split으로 열기 : **ctrl + k, v**
